# GreenStack: Decentralized Carbon Credit Marketplace

GreenStack is a platform that tokenizes and trades carbon credits on the Stacks blockchain, leveraging Bitcoin's security and Stacks' smart contract capabilities to create a transparent, efficient market for carbon offsets.

## Project Structure

- `/frontend`: React application for the user interface
- `/contracts`: Clarity smart contracts

## Setup

1. Clone the repository
2. Install dependencies:
   ```
   cd frontend
   npm install
   ```
3. Run the frontend:
   ```
   npm start
   ```
4. For contract development, use Clarinet in the `/contracts/greenstack` directory

## Features

- Tokenized Carbon Credits
- Verified Project Listings
- Automated Verification
- Fractional Ownership
- Integration with Bitcoin
- Dashboard for Carbon Impact
- Smart Contract-based Retirement
